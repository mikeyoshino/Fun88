Game Player Page — Iframe Embed (Critical Details)
The iframe src is built server-side (never hardcoded inline):

  https://html5.gamedistribution.com/{game.ProviderGameId}/
    ?gd_sdk_referrer_url=https://yoursite.com/games/{game.Slug}
    &gdpr-tracking={consentTracking}       ← from user's cookie consent
    &gdpr-targeting={consentTargeting}     ← from user's cookie consent
    &gdpr-third-party={consentThirdParty}  ← from user's cookie consent

Built by: GdEmbedUrlBuilder (typed service, no string concatenation in views)

GDPR consent state:
  • Stored in a separate HttpOnly consent cookie
  • Set when user accepts/rejects cookie banner on first visit
  • Read server-side when rendering the game player page
  • 0 = rejected, 1 = accepted — defaults to 0 (privacy-first)

For custom/own games:
  • game_url is the raw iframe src or full embed HTML stored at import
  • No GD parameters needed — rendered as-is
GD SDK postMessage Events (page must handle these)
GD's SDK inside the game fires postMessage events to the parent page.
Our game player page listens and responds:

  SDK_GAME_START     → game loaded, remove loading overlay
  SDK_GAME_PAUSE     → ad is about to show, blur/mute game controls
  SDK_GAME_RESUME    → ad finished, restore game controls
  SDK_ERROR          → game failed to load, show error state

Handled in: /wwwroot/js/game-player.js  (vanilla JS, no framework deps)
No inline event logic in Razor views — all in the dedicated script file.

Config — Strongly Typed, Never Magic Strings

class GameDistributionOptions          // bound from appsettings.json
{
    public string PublisherApiKey { get; init; }
    public string PublisherId { get; init; }
    public string CatalogApiBaseUrl { get; init; }  // "https://catalog.gamedistribution.com"
    public string EmbedBaseUrl { get; init; }       // "https://html5.gamedistribution.com"
    public int CatalogPageSize { get; init; }       // default: 100
}

class GdprConsentOptions
{
    public string CookieName { get; init; }         // "gdpr_consent"
    public int CookieExpiryDays { get; init; }      // 365
}
