Layout Rules Captured

Top Nav

100% viewport width. Sits above sidebar + content. Contains: hamburger · logo · search · bell · heart · avatar · login button. Gradient: #1e1a2e → #16132a

Sidebar

Sits below nav. Collapsed: 60px (icon only). Expanded on hover: 220px (icon + label). CSS transition only — no JS. Active item: purple left border + #a29bfe text. Icons: monochrome, #7c7a96 inactive.

Sidebar sections

Top: quick links (Home, Recently played, New, Popular, Updated, Multiplayer, Leaderboards). Divider. Bottom: all categories A–Z, scrollable.

Hamburger

☰ in top nav toggles sidebar between collapsed/pinned-open state. Default: collapsed (icon only). Hover always expands. Click to pin open.
