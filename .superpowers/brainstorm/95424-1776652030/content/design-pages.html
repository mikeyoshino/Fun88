Language Switching
Language detected in order:
  1. Authenticated user's preferred_language from DB
  2. "lang" cookie (set when user clicks EN/TH switcher)
  3. Accept-Language request header
  4. Default: "en"

GET /language/set?lang=th&returnUrl=/games/online-strike-assault
  → Sets lang cookie (HttpOnly: false — JS can read for UI)
  → Redirects to returnUrl
  → If logged in: also updates users.preferred_language in DB

URL structure: language is NOT in the URL path (/th/games/...)
  → Simpler routing, language resolved per-request from cookie/header
